# まえがき

本書『AWS経験者のためのAzure実践入門【共通基礎編】』を手に取っていただき、ありがとうございます。

本書は、AWS(Amazon Web Services)で一通りの環境構築を経験したエンジニアが、Azure の土台——ID・ガバナンス・ネットワーク・監視・IaC——をハンズオン形式で身につけるための本です。単なるサービスの対応表ではなく、「なぜ Azure はこう設計されているのか」を AWS との対比を通じて理解し、最終的に **ミニランディングゾーン** と呼べる再現可能な土台を組み上げることをゴールとします。

## 本書の対象読者

- AWS でアカウント設計・IAM・VPC・監視の構築を経験したことがある方
- これから Azure を業務や個人プロジェクトで使う必要がある方
- 「Azure のサービス名は聞いたことがあるが、AWS の知識をどうマッピングすればよいか分からない」と感じている方

## 本書の進め方

各章は次の 5 部構成で統一しています。

1. このリソースは何か
2. AWS との対比
3. 土台への適用(Before / After)
4. 設計の勘所
5. ハンズオン
